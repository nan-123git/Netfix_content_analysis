## 🔹 Overview
This project analyzes Netflix content data to uncover trends in content growth, audience targeting, and global distribution.

The objective is to transform raw data into actionable business insights that support strategic decision-making in content investment and platform expansion.

---

## 🔹 Problem Statement
With increasing competition in the streaming industry, understanding content strategy is critical.

This project explores:
- Evolution of Netflix content over time  
- Distribution between Movies and TV Shows  
- Audience targeting through content ratings  
- Global content production patterns

## 🔹 Project Workflow

### 1. Data Cleaning (Python)
- Handled missing values and removed inconsistencies  
- Processed and standardized duration into usable formats  
- Cleaned and structured dataset for analysis  
- Performed feature engineering (e.g., extracting year, categorizing duration)  
- Converted data types for accurate analysis  
- Removed duplicates and irrelevant records  
- Resolved data import and formatting issues by handling inconsistencies in the raw dataset  

### 2. Data Analysis (SQL)
- Analyzed country-wise content distribution  
- Performed rating frequency analysis  
- Conducted trend analysis over years  
- Used GROUP BY and aggregation functions for insights  
- Applied JOIN operations to combine and analyze related data  
- Identified top-performing countries, ratings, and content types  

### 3. Data Visualization (Power BI)
- Designed KPI cards for key business metrics  
- Built line chart to analyze content growth trends  
- Created bar charts for country-wise and rating distribution  
- Developed pie chart for content mix (Movies vs TV Shows)  
- Implemented map visualization for global content distribution  
- Added slicers to enable dynamic filtering by country and multi-country selection, allowing full dashboard interactivity  
- Designed DAX measures to handle missing values and replace blank outputs with zero for improved data clarity  
- Created dynamic insight text using DAX measures to reflect real-time changes based on user interaction  
- Enabled interactive exploration through cross-filtering across all visuals

## 🔹 Key Insights
- Content growth accelerated significantly after 2015, peaking around 2019–2020  
- Platform is heavily movie-driven (~69%), indicating a strong film-focused strategy  
- Content is primarily targeted toward mature audiences (TV-MA dominant)  
- Content production is concentrated in the United States, with growing contributions from India and the UK  

---
## 🔹 Business Insight Approach

This project goes beyond data visualization by focusing on how insights can support real business decisions.

Instead of only presenting trends, the analysis is structured to answer:
- What is happening in the data  
- Why these patterns exist  
- How they impact business strategy  

The goal is to bridge the gap between data analysis and decision-making by translating findings into actionable insights.

## 🔹 Business Recommendations
- Shift from volume-driven to quality-focused content strategy  
- Increase investment in TV Shows to improve long-term user engagement  
- Diversify content ratings to expand audience reach  
- Strengthen presence in emerging markets to enhance global footprint
